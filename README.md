# grimoire

◼ Store prompts. ◻ Edit them. △ Copy to clipboard.

---

A minimal CLI for managing your prompt library.

## Install

```bash
bun install -g grimoire
```

## Usage

```
grimoire              # interactive mode
grimoire add          # add new prompt
grimoire list         # list all prompts
grimoire edit <name>  # edit a prompt
grimoire copy <name>  # copy to clipboard
grimoire rm <name>    # delete a prompt
```

## Storage

Prompts live in `~/.grimoire/prompts.json`
